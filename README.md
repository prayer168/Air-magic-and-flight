# 空氣魔法與飛行互動實驗室 Air Magic and Flight Lab

適用國小三年級到國中的空氣與飛行互動數位教材。單一 `index.html`，可直接開啟、無需後端或登入。

## 16 章、10 個互動實驗

1. 首頁與任務說明　2. 空氣看不見，真的存在嗎？
3. 紙飛機設計　4. 安全空氣吹管　5. 氣球火箭　6. 空氣砲
7. 降落傘　8. 紙飛旋翼　9. 伯努利漂浮球　10. 飛行種子與仿生
11. 風帆車競速　12. 機翼升力
13. 真實實驗操作指南　14. 探究紀錄表　15. 生活應用與迷思澄清　16. 學習挑戰與成果摘要

每個實驗都採**探究流程**：情境問題 → 預測 → 選變因 → 公平測試 → 啟動模擬 → 觀察 → 記錄數據 → 做三次 → 比較 → 寫結論 → 改進設計。同時改變多個變因會跳出公平測試提醒。

## 主要功能

- **10 個可操作模擬**：每個 6–7 個具名拉桿／選單（含名稱、數值、單位與影響說明），即時預覽、動畫觀察、自動記錄數據表。
- **分級學習**：基礎模式（國小，大按鈕、白話說明）／進階模式（國中，數值、受力分析、科學解釋）一鍵切換。
- **探究紀錄表**：11 欄位＋數據表，自動產生長條圖／折線圖，可儲存、下載文字檔、複製、列印；用 LocalStorage 保存進度與紀錄。
- **10 張翻面迷思卡**、**8 則生活應用**、**15 題即時回饋評量**（附原因）＋研究員證書。
- **無障礙與工具**：固定導覽列、章節切換、學習進度條、深／淺色模式、全螢幕、音效、動畫速度、減少動態、鍵盤操作與焦點提示、語意化 HTML 與 ARIA。

## 科學正確性

- 升力不只用「等時通過」解釋，強調氣流向下偏轉的反作用力與壓力差，並示範失速。
- 氣球火箭為作用力與反作用力；空氣砲呈現渦環而非固體球；漂浮球涉及壓力差、氣流偏轉與穩定平衡。
- 所有模擬數值標示為「模擬結果」（簡化模型），真實結果受材料、製作精度、環境風與測量誤差影響。來源查核見 `docs/references.md`。
