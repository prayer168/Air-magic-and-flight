# 空氣魔法與飛行 Air Magic and Flight

國小自然科互動數位教材。透過四個「空氣魔法」實驗，探索看不見的空氣如何推動物體：

| 單元 | 核心概念 |
|---|---|
| ✈️ 紙飛機設計 | 飛行四力：升力、重力、推力、阻力；機翼設計與投擲角度 |
| 🎯 吹箭 | 空氣壓力差推動；管長越長、加速距離越長、出口速度越快 |
| 🎈 氣球火箭 | 牛頓第三運動定律：作用力與反作用力 |
| 💨 空氣砲 | 空氣佔有空間、能推動物體；渦環的形成與圓孔／方孔比較 |

另含 13 題即時回饋的「魔法闖關」測驗與查核過的延伸學習資源。

## 使用

- 線上版：https://prayer168.github.io/Air-magic-and-flight/
- 本機：純靜態 HTML/CSS/JS，直接開啟 `index.html`，或 `python -m http.server` 後瀏覽。

## 特色

- 四個互動模擬（操作變因 → 觀察結果 → 思考題），全部為自製 SVG 動畫。
- 支援滑鼠、觸控、鍵盤（頁籤可用方向鍵切換）與 `prefers-reduced-motion`。
- 模擬數值為教學示意，鼓勵學生實際動手驗證；來源查核見 `docs/references.md`。

## 專案結構

```
index.html      七頁籤教材主體（含四張教學示意 SVG）
css/style.css   樣式與響應式、無障礙設定
js/app.js       頁籤系統、四個模擬、闖關測驗
docs/           參考來源與測試報告
img/og/         社群分享預覽圖
```
